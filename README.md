# 🐰 Talking Rabbitt – Conversational Business Intelligence

Talking Rabbitt is a conversational intelligence layer built on top of structured business data.  
Instead of navigating dashboards, leaders can directly **ask business questions** and receive executive-level insights with contextual visualizations.

## 🎯 Problem Statement

Traditional BI tools (PowerBI, Tableau) require:
- Dashboard creation
- Manual filtering
- Analyst dependency
- Technical expertise

Talking Rabbitt eliminates dashboard friction by enabling:

> “Talk to your business. Get insights instantly.”

---

## 🧠 Core Value Proposition

- Upload structured CSV data
- Ask natural language business questions
- Receive executive-level insights
- Automatically generate contextual charts
- Fallback intelligent logic if AI model is unavailable

---

## 📊 Features

- ✅ CSV Upload
- ✅ Conversational Query Interface
- ✅ Executive-Level Insight Generation
- ✅ Smart Intent-Based Visualization
- ✅ Local Analytics Fallback Engine
- ✅ Secure Environment Variable Handling

## 🛠 Tech Stack

- **Streamlit** – Frontend & UI
- **Pandas** – Data processing
- **Matplotlib** – Visualization
- **HuggingFace Inference API (Optional)** – LLM-powered insights
- **python-dotenv** – Secure environment variable handling
